4. IIFE (Immediately Invoked Function Expression)

Function that runs automatically when page loads
Executes immediately without being called
Syntax:
(function(parameters) {
  // code
})(arguments);

Example 1: Simple IIFE

(function(a, b) {
  console.log(a + b);
})(5, 19);
// Output: 24 (runs immediately)
4

Example 2: IIFE with Return Value

var result = (function(a, b) {
  return (a + b);
})(5, 18);
console.log("Sum of two numbers is: " + result);
// Output: Sum of two numbers is: 23
‚òÖ IIFE Use Cases:
‚Ä¢ Avoid polluting global scope
‚Ä¢ Execute code immediately
‚Ä¢ Create private variables
‚Ä¢ Module pattern

Arrow Functions (ES6)

Introduced in ECMAScript 2015 (ES6)
Shorter syntax for writing functions
Syntax:
const functionName = (parameters) => {
  // code
  return value;
}

Basic Arrow Function

const sum = (a, b) => {
  return a + b;
}
console.log(sum(5, 6)); // 11

Arrow Function - Short Syntax

‚òÖ Rules for Short Syntax:
Rule 1: If function body has only one expression,
you can remove { } braces and return keyword

Example:

const sum = (a, b) => `Sum of ${a} and ${b} = ${a + b}`;
console.log(sum(5, 6));
// Output: Sum of 5 and 6 = 11
6
Rule 2: If only one parameter,
you can remove ( ) parentheses around parameter

Example:

const square = a => `Square of ${a} = ${a * a}`;
console.log(square(5));
// Output: Square of 5 = 25
Rule 3: If no parameters,
you must use ( ) empty parentheses

Example:

const greet = () => console.log("Hello world");
greet();
// Output: Hello world

Arrow Function Examples

Example 1: Multiple Parameters

const sum = (a, b) => `Sum of ${a} and ${b} = ${a + b}`;
console.log(sum(5, 6));

Example 2: Single Parameter

const square = a => `Square of ${a} = ${a * a}`;
console.log(square(5));

Example 3: No Parameters

const greet = () => console.log("Hello world");
greet();

Key Points for Exams

‚òÖ Function Basics:
‚Ä¢ Functions are reusable blocks of code
‚Ä¢ Can have parameters (inputs)
‚Ä¢ Can return values (outputs)
‚Ä¢ Declare before calling (best practice)
‚òÖ Function Types:
1. Declaration - function name() { }
2. Expression - var x = function() { }
3. Anonymous - function() { }
4. IIFE - (function() { })()
5. Arrow - () => { }
‚òÖ Arrow Function Rules:
‚Ä¢ One expression ‚Üí no { } or return needed
‚Ä¢ One parameter ‚Üí no ( ) needed
‚Ä¢ No parameters ‚Üí ( ) required
‚òÖ IIFE Characteristics:
‚Ä¢ Runs immediately
‚Ä¢ Creates private scope
‚Ä¢ Syntax: (function() { })(arguments)

Practice Questions

Q1: Create function to check if number is even
