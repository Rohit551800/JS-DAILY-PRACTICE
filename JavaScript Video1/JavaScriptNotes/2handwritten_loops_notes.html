Loops in JavaScript

Loops are used to execute a block of code repeatedly
until a specified condition is met
Types of Loops:
1. while loop
2. do-while loop
3. for loop

1. While Loop

Syntax:
while(condition) {
  // code to execute
  // update expression
}
‚Ä¢ Checks condition before executing code
‚Ä¢ May not execute even once if condition is false initially

Example:

let i = 0;
while(i < 10) {
  console.log("Hellow world");
  i++;
}
// Prints "Hellow world" 10 times
1

2. Do-While Loop

Syntax:
do {
  // code to execute
  // update expression
} while(condition);
‚Ä¢ Checks condition after executing code
‚Ä¢ Executes at least once even if condition is false

Example:

let j = 0;
do {
  console.log("Love you muski ‚ù§Ô∏è");
  j++;
} while(j < 5);
// Prints "Love you muski ‚ù§Ô∏è" 5 times
‚òÖ Key Difference:
‚Ä¢ while ‚Üí checks first, then executes
‚Ä¢ do-while ‚Üí executes first, then checks
2

3. For Loop

Syntax:
for(initialization; condition; update) {
  // code to execute
}
Three Parts:
1. Initialization - runs once at start
2. Condition - checked before each iteration
3. Update - runs after each iteration

Example:

for(let x = 0; x < 5; x++) {
  console.log("Muskan ‚ù§Ô∏è");
}
// Prints "Muskan ‚ù§Ô∏è" 5 times

Problem 5: Half Pyramid Pattern

let x = 5;
for(let i = 1; i <= x; i++) {
  let row = "";
  for(let j = 0; j < i; j++) {
    row += "* ";
  }
  console.log(row);
}
Output:
*
* *
* * *
* * * *
* * * * *
How it works:
‚Ä¢ Outer loop (i) controls rows (1 to 5)
‚Ä¢ Inner loop (j) prints stars (0 to i-1)
‚Ä¢ Each row has i number of stars
‚òÖ Nested Loop Concept:
Outer loop runs 5 times
For each outer loop, inner loop runs i times

Loop Control Statements

1. break Statement

Immediately exits the loop

Example:

for(let i = 1; i <= 10; i++) {
  if(i == 5) {
    break; // stops loop when i = 5
  }
  console.log(i);
}
// Output: 1 2 3 4

2. continue Statement

Skips the current iteration and continues with next

Example:

for(let i = 1; i <= 5; i++) {
  if(i == 3) {
    continue; // skips 3
  }
  console.log(i);
}
// Output: 1 2 4 5

When to Use Which Loop?

for loop

‚Ä¢ When you know the exact number of iterations
‚Ä¢ Best for counting and iterating arrays
Example: Print 1 to 10

while loop

‚Ä¢ When you don't know exact iterations
‚Ä¢ Condition-based repetition
Example: Keep asking until valid input

do-while loop

‚Ä¢ When code must execute at least once
‚Ä¢ User input validation
Example: Menu-driven programs

Common Mistakes to Avoid

‚ùå Infinite Loop:
Forgetting to update loop variable
let i = 0;
while(i < 10) {
  console.log(i);
  // forgot i++; ‚Üí infinite loop!
}

Key Points for Exams

‚òÖ Loop Comparison:
for loop:
‚Ä¢ Best for known iterations
‚Ä¢ Syntax: for(init; condition; update)
‚Ä¢ Example: for(let i=0; i<10; i++)
while loop:
‚Ä¢ Checks condition before execution
‚Ä¢ May not execute at all
‚Ä¢ Example: while(i < 10)
do-while loop:
‚Ä¢ Checks condition after execution
‚Ä¢ Executes at least once
‚Ä¢ Example: do { } while(i < 10);
‚òÖ Control Statements:
‚Ä¢ break ‚Üí exits loop completely
‚Ä¢ continue ‚Üí skips current iteration
‚òÖ Nested Loops:
Used for patterns, 2D arrays, matrices
Outer loop controls rows, inner loop controls columns
