Operators & Type Conversion

typeof Operator

Used to check the data type of a variable
var myName = "Rohit";
console.log(myName);
console.log(typeof myName);
// Output: string

Type Conversion

Convert String to Number

let str = "hello";
console.log(typeof Number(str));
// Output: number (NaN)

OR (using unary +)

console.log(typeof +str);
// Output: number

Convert Number to String

let x = 5;
console.log(typeof String(x)); // Method 1
console.log(typeof (x + "")); // Method 2
// Output: string

Logical Operators

1. && (AND Operator)

Returns true when both expressions are true
console.log(5 > 2 && 5 > 6);
// Output: false (1 condition is false)
console.log(5 > 2 && 5 > 3);
// Output: true (both conditions are true)

2. || (OR Operator)

Returns true when at least one condition is true
console.log(5 > 2 || 5 < 3);
// Output: true (1 condition is true)

3. ! (NOT Operator)

Returns true if condition is false and vice-versa
var isOpen = false;
console.log(!isOpen);
// Output: true

Tricky Questions

Q1: What will be the output?

console.log(typeof("5" - 3));
// Output: number
'-' operator converts "5" to number, then 5 - 3 = 2

Q2: What will be the output?

console.log(2 < 12 < 5);
// Output: true
Step 1: 2 < 12 = true (true = 1)
Step 2: 1 < 5 = true

Q3: What will be the output?

console.log("20" + 10 + 10);
// Output: 201010 (string)
"20" + 10 = "2010", then "2010" + 10 = "201010"

Operator Precedence (High to Low):

1. () - Parentheses
2. !, ++ , -- - Unary operators
3. *, /, % - Multiplication, Division, Modulus
4. +, - - Addition, Subtraction
5. <, >, <=, >= - Comparison
6. ==, === - Equality
7. && - Logical AND
8. || - Logical OR
9. ?: - Ternary
10. = - Assignment
