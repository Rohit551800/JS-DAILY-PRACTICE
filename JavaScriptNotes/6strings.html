Strings in JavaScript

Strings can be created by single or double quotes
• Single quotes → used with single character
• Double quotes → used with multiple characters

Indexing: 0 to string.length - 1

String Length

String_name.length is used to get the total length of a string
Space is also calculated as length

Example:

let str = "Hellow world";
console.log(str.length);
// Output: 12

Example:

let str = "This is Rohit\'s \"Welcome myself\" book";

String Searching

1. indexOf()

Returns the first occurrence of the string in a string
Returns -1 if not present
Strings are CASE SENSITIVE
console.log(string.indexOf('Singh'));
2

Converting String into Array

let string = "Rohit Singh Gouria";
let str = Array.from(string);
console.log(str);

Storing and printing string elements with indices:

const strMap = str.map((curElem, index) => {
  return `${curElem} - ${index}`;
});
console.log(strMap);

2. lastIndexOf()

let str = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(str.length);
console.log(str.lastIndexOf("JavaScript", 40));
40 dalne k baad backward direction mai move karega and last character ko find karega
Output: 6

3. search()

Search the string from starting
If found → return index number
Else → return -1
Best thing: Can use REGEX
3

Regex: Case-sensitive characters can be accessed

Syntax: string_name.search(/string_to_be_check/i);

Example:

let str = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(str.search(/JavaScript/i));

4. match()

let str = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(str.match("Javascript")); // null (case-sensitive)
console.log(str.match("JavaScript")); // returns data
Returns data in array format:
[
  'JavaScript',
  index: 6,
  input: 'Hello JavaScript, welcome to out world...',
  groups: undefined
]
Behind the scene: match() converts "JavaScript" to regular expression
i.e., /JavaScript/i at global level
4

To get number of occurrences:

console.log(str.match(/JavaScript/g)); // matches at global level
// Output: ['JavaScript', 'JavaScript']

5. matchAll()

Returns an iterator of all matches
Provides detailed information about each match
Returns empty iterator if no match found
let str = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(str.matchAll("Javascript")); // no result
// Output: Object [RegExp String Iterator] {}
matchAll() converts string to regular expression at global level
i.e., string_name.matchAll(/Pattern/g);
let result = str.matchAll("JavaScript");
Provides output as iterator → Need to use loop to get values
• Loops can be used
• Spread operator can be used
5

Using For Loop:

for(let i of result) {
  console.log(i); // console.log(i[0]); for 1st value
}

Using Spread Operator:

console.log(...result);
Output:
[
  'JavaScript',
  index: 6,
  input: 'Hello JavaScript, welcome to out world...',
  groups: undefined
]
[
  'JavaScript',
  index: 45,
  input: 'Hello JavaScript, welcome to out world...',
  groups: undefined
]

includes()

Returns true or false
Regular expression CANNOT be used
let text = "Hello JavaScript, welcome to out world best JavaScript course";
let includeResult = text.includes("JavaScript");
console.log(includeResult); // true
6

startsWith()

Returns true or false
let text = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(text.startsWith("Hello JavaScript"));
// Output: true

endsWith()

Returns true or false
let text = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(text.endsWith("urse"));
// Output: true

String Extraction

slice()

Extracts a part of a string
Returns the extracted part in a new string
let text = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(text.slice(6, 17));
// Output: JavaScript
7

substring()

DEPRECATED
Extracts a portion of string based on starting and ending indices
Same as slice()

replace()

Used to replace the string with any value
Regex can be used

Example:

let text = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(text.replace("H", "Rohit"));
Agar har jagah element ki occurrence ko change krna hai toh
replaceAll() is used

Example:

console.log(text.replaceAll("JavaScript", "Rohit"));

Extracting String Characters

charAt()

let text = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(text.charAt(4));
// Output: "o"
If didn't get any value, returns empty string
8

charCodeAt()

Returns ASCII value of character
let text = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(text.charCodeAt(7));
// Output: 97 (ASCII value of 'a')

at()

On negative value, indexing starts from right side
From -1 to -n
let text = "Hello JavaScript, welcome to out world best JavaScript course";
console.log(text.at(-2));

trim()

Used to remove extra space from the string
let text = "          hello, world      ";
let result = console.log(text.trim());

split(), reverse(), join()

split() → Converts string to array
reverse() → Reverses the array
join() → Converts array back to string

const result1 = () => {

  }

Q2: Count number of vowels in a string

const result2 = (s) => {
  let vowels = 'aeiou';
  let counter = 0;
  for(let i of s) {
    if(vowels.includes(i)) counter++;
  }
  return counter;
}
console.log(result2("abcdefghijklmnopqrstuvwxwzaaaaaa"));

Q3: Check if all vowels present in string

const result3 = (s) => {
  let vowels = "aeiou";
  for(let i of vowels) {
    if(!s.includes(i)) {
      return false;
    }
  }
  return true;
}
console.log(result3("a b c e i o y u"));

Q4: Check if string is Pangram

String: "the quick brown fox jumps over the lazy dog"
Pangram = Contains all 26 letters of alphabet

Method 1: Using Loop

const result = (str) => {
  let totalChar = "";
  for(let i = 97; i <= 122; i++) {
    totalChar += String.fromCharCode(i);
  }
  console.log(totalChar);
  for(let i of str) {
    if(i != " " && !totalChar.includes(i)) return false;
  }
  return true;
}
11

Method 2: Using Set (Better Solution)

const result = (str) => {
  const set = new Set();
  for(let i of str.toLowerCase()) {
    if(i >= 'a' && i <= 'z') {
      set.add(i);
    }
  }
  return set.size === 26;
}
console.log(result("the quick brown fox jumps over the lazy dog"));

Key Points for Exams

★ Important Methods:
• indexOf() → first occurrence
• lastIndexOf() → last occurrence
• includes() → true/false (no regex)
• search() → can use regex
• match() → returns array
• matchAll() → returns iterator
• slice() → extracts part
• replace()/replaceAll() → replace strings
• charAt() → character at index
• charCodeAt() → ASCII value
• trim() → removes spaces
• split() → string to array
